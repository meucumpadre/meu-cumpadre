## O Estado Inicial — Diagnóstico da Crise

A HP ENVY 15 x360 chegou ao limite:

- **5 GB livres** num SSD de 223 GB — o sistema operava em colapso silencioso
- **1,2 GB de RAM livre** de 16 GB — cada processo disputava oxigênio
- **Triple sync ativo:** Google Drive + iCloud + OneDrive — três entidades drenando energia simultaneamente
- **CapCut com 32 GB** de cache acumulado — um parasita invisível
- **i5-5200U de 2015** operando no plano "Equilibrado" — a CPU se autossabotando

A máquina não era lenta. A máquina estava sendo sufocada.

## O Exorcismo — O que foi removido

| Exorcizado | Liberado |
|---|---|
| Temp do Windows | ~13 GB |
| CapCut (cache + app) | ~30 GB |
| Cache Chrome | ~6 GB |
| Whisper duplicado | ~1,5 GB |
| Wondershare installer | ~0,2 GB |
| Conta Drive antiga (`contato@aposentadorianamao.com.br`) | resíduo histórico |

| OneDrive | nunca existiu — fantasma no registro |

**Total recuperado: ~50 GB. De 5 GB para 55 GB livres.**

## A Ressurreição — O que foi construído

### Infraestrutura

- **Plano de energia: Alto Desempenho** — a CPU parou de se segurar
- **CLAUDE_CODE_USE_POWERSHELL_TOOL=1** — fim da dependência do Git Bash
- **Git inicializado em 2 workspaces:**
  - `C:\Users\HP` — 18 arquivos, commit `0e5c03b`
  - Vault Obsidian — 469 arquivos, 52.663 linhas, commit `4cc18a8` via `--separate-git-dir`
- **`C:\Dev` criado** — projetos de código saem do Drive, I/O duplo eliminado
  - `clickup-obsidian-sync` migrado
  - `llama-cookbook` migrado

### Cowork Armado

- Conectores ativos: **Filesystem · PDF Tools · Control Chrome · Gmail · Calendar · ClickUp · Figma · Minutes**
- Windows-MCP: batalha épica — `spawn uv ENOENT` vencido no PATH, derrotado no timeout. **Adiado para o Mac Mini.**
- N8N: aguarda senha Bravy
