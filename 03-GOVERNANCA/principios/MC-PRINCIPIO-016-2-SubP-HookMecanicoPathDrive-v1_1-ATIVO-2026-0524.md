# Sub-Princípio MC #16.2 PROPOSTO

## §1 · Enunciado

Sub-princípio #2.1 ATIVO (MB Autocontido · Conteúdo Inline) não opera apenas por consciência verbal do cofounder C3.1. Requer **hook mecânico operacional** que torna a violação estatisticamente inviável.

3 caminhos válidos para honrar #2.1 sem violação:

### Caminho A · Inline Literal (conteúdo completo dentro do MB)

Cofounder cola o conteúdo bit-a-bit do arquivo a ser persistido **dentro do MB**, entre marcadores demarcados (```markdown ... ``` ou ````markdown ... ````). MB fica grande mas autocontido. Code DELL copia-cola direto.

**Quando usar:** arquivos pequenos a médios (até ~50KB) · cunhagens críticas onde audit trail inline é exigência.

### Caminho B · Path-Drive com Hash + Bytes Declarados

Cofounder produz o arquivo em `/home/claude/` · founder Alessandro faz upload manual no Drive (MB-040 ou pasta canônica) · MB declara: path Drive exato + hash SHA-256 + tamanho em bytes esperados + comportamento de espera do Code DELL.

**Quando usar:** arquivos médios a grandes (>50KB) · arquivos binários (PDFs, PNGs, RAR) · workflow estabelecido onde founder está ativo na sessão.

### Caminho C · Auto-MCP-Drive (futuro)

Quando cofounder C3.1 ganhar acesso direto ao Drive via MCP (Movimento futuro · não-disponível em 24/05/2026), persistirá direto sem ponte humana.

**Quando usar:** futuro · ainda não-operacional.

## §2 · Hook mecânico cofounder (regra inviolável)

**Antes de declarar funcional qualquer MB que persista artefato canônico**, cofounder C3.1 deve responder explicitamente no próprio MB:

```
Caminho declarado: [A · INLINE LITERAL] ou [B · PATH-DRIVE] ou [C · AUTO-MCP]
Conteúdo:
  - Se A: bloco demarcado com conteúdo completo bit-a-bit
  - Se B: path Drive exato + hash SHA-256 + bytes + ação do founder declarada
  - Se C: indisponível em 24/05/2026
```

Sem essa declaração explícita · MB não é autocontido · violação automática de #2.1.

## §3 · Sanção por violação

Mesma sanção #16.1: retrabalho cofounder + errata documentada + custo carregado pelo C3.1 (não terceirizado).
