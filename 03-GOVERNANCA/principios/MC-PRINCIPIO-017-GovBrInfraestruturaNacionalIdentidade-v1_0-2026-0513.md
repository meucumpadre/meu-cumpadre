# Princípio MC #17 — gov.br = infraestrutura nacional de identidade digital

## Enunciado
> O acesso gov.br é a identidade BR de TODO fluxo digital do cidadão (CNH, Shopee/Correios, INSS, CadÚnico, Detran, bancos). Hook 0 MC é proto-MVP de "API Humana Identidade Digital BR" cross-vertical. Resolver gov.br para o hipervulnerável = abrir simultaneamente N portas que ele não consegue abrir sozinho.

## Origem
Cunhado em 12/05/2026 como insight estratégico. Reconhecimento de que Hook 0, originalmente concebido como pré-RouterEthics INSS-only, é arquitetonicamente cross-vertical.

## Aplicação
1. Hook 0 atual: classifica nível Prata/Ouro/Bronze + 18 cenários · pré-RouterEthics
2. Cross-vertical futuro: mesma stack serve CadÚnico, Detran, Correios, bancos (Fase 2+)
3. ADR-016 (futuro, pós-Juliana): Stack Hook 0 API Humana Identidade Digital BR
4. FINEP/BNDES citável: TA + IA + impacto + interoperabilidade gov

## Implicações estratégicas
- Hook 0 = moat institucional (22 anos de Alessandro lendo INSS aplicado a gov.br completo)
- Cross-vertical = expansão horizontal sem nova vertical jurídica
- Compliance: vetar Cenário B (operar conta gov.br do cidadão) em TODOS os verticais
