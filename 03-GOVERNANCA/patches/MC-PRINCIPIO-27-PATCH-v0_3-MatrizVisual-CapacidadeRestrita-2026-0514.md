## 2 · TEXTO CANÔNICO ATUALIZADO

> **Princípio #27 v0.3 — Matriz Visual Multi-Camada com Capacidade Restrita.**
>
> Toda produção visual MC sob a Cascata #28 obedece uma matriz tripartida que distingue laboratórios canônicos de ferramentas de aplicação (Princípio #30):
>
> **Camada C5 — Laboratório Canônico Fundacional (manual founder):**
> - Plataforma: `claude.ai/design`
> - Operador: founder Alessandro manual
> - Output: Brand Foundation imagética + variações derivadas com vero-semelhança ao Seed MC
> - Autorizada para: Tier 1 (Principal, Marca) + Tier 3 (variantes temáticas) + Camada 3b (variação derivada do símbolo Seed)
>
> **Camada C5b — Laboratório Canônico Externo (validado):**
> - Plataforma: Manus.im com skill `contemplative-brand-identity`
> - Operador: founder Alessandro ou cofounder C1 (via prompts canônicos `successful_prompts.md` §3 do SPEC v1.0)
> - Output: idem C5 com validação empírica (produziu os 4 PNGs canônicos selados 13/05/2026)
> - Autorizada para: idem C5
>
> **Camada C2-Visual — Ferramenta de Aplicação (Cowork DELL):**
> - Plataformas: Canva MCP, Figma MCP, Adobe MCPs
> - Operador: Cowork autônomo via MCPs
> - Output: aplicação direta de PNG canônico como bloco fixo em peças derivadas
> - Autorizada APENAS para: Camada 3a (aplicação direta — crop, composição com PNG fixo + texto vetorial, diagramação)
> - **VEDADA para:** Camada 3b (variação derivada do símbolo Seed, regeneração de Tier 1/3, iconografia derivada, hero slides com Seed modulado, highlights temáticos com símbolo redesenhado)
>
> **Regra inviolável:** Cowork antes de qualquer chamada Canva/Figma/Adobe MCP visual confirma: *"Esta peça é Camada 3a (uso de PNG canônico como bloco fixo)?"* Se a resposta for NÃO ou DUVIDOSA, a peça vai para C5/C5b (founder/cofounder produz manualmente em Laboratório Canônico).

## 4 · CHECKLIST PRÉ-CANVA/FIGMA/ADOBE MCP (obrigatório Cowork)

Antes de qualquer chamada Canva/Figma/Adobe MCP visual, Cowork confirma no log:

```
[ ] Esta peça USA um dos 4 PNGs canônicos como bloco visual fixo? (SE NÃO → ABORTAR + escalar para founder)
[ ] Esta peça NÃO requer regeneração ou modulação do símbolo Seed? (SE REQUER → ABORTAR + escalar)
[ ] Esta peça é diagramação textual + PNG fixo + crop/composição básica? (SE NÃO → revisar escopo)
[ ] Posso entregar com vero-semelhança ao Seed sem que a plataforma reinterprete arte? (SE NÃO → ABORTAR)

SE 4/4 ✅ → prosseguir Camada 3a
SE qualquer ❌ → escalar para C5/C5b (founder/cofounder Laboratório Canônico)
```
