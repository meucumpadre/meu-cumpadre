# meu-cumpadre/design-system

> Design System Meu Cumpadre · DNA institucional portátil para AI agents (Claude Design, Claude Code, Cursor, v0).

## O que tem aqui

- `MC-DESIGN.md` — canon textual mestre (este arquivo é a alma do sistema)
- `tokens/colors.css` — tokens cromáticos canônicos (gerar a partir de §4 do DESIGN.md)
- `tokens/typography.css` — tokens tipográficos (gerar a partir de §5)
- `tokens/spacing.css` — tokens de espaçamento (gerar a partir de §7)
- `assets/marca-selada/` — 3 ativos visuais canônicos da marca (16/05/2026)
- `assets/foundation-v0_3/` — HTML + PDF do Foundation fundacional
- `SKILL.md` — skill portátil para Claude Code (gerada pelo Sistema de Projeto MC)

## Como usar (humanos)

1. Leia `MC-DESIGN.md` completo
2. Use tokens CSS em qualquer projeto frontend
3. Respeite hierarquia D > C > V
4. Honre vetos absolutos da §9
