## §26 · TRAVA CASCATA VERBO→IMAGEM (NOVO v1.4 — Princípio #28)

### Princípio operacional inviolável

**Cowork NÃO INICIA produção visual em massa (C2-Visual via Canva/Figma/Adobe MCPs) ENQUANTO as 3 camadas anteriores não estiverem cumpridas:**

| Camada | Status verificável | Quem produz |
|---|---|---|
| 0 — Vault canônico organizado + Documentação canônica MC estruturada por pilar | `_INDEX.md` em `principios/` reflete 28 princípios + `MC-MAPA-Documentacao-vX_Y.md` existente no Vault | Code DELL + Cowork (C2-Op) |
| 1 — Documentação de Instrução de Marca | `MC-DOCUMENTACAO-INSTRUCAO-MARCA-vX_Y.md` no Vault em `05-ESTRATEGIA/manifestos/` ou `01-IDENTIDADE/` | C1 (Claude.ai cofounder) + Founder valida |
| 2 — Brand Foundation imagética refletida | Pasta `Documentação\Identidade Visual\01-Brand-Foundation-v1_0\` com ativos paleta HEX + tipografia + logos | Founder manual em claude.ai/design (C5) |

### Pre-flight obrigatório de QUALQUER MB visual em massa recebido por Cowork

```
1. VERIFICAR _INDEX.md em principios/ reflete 28 princípios consolidados → SE NÃO: ABORTAR + sinalizar AGUARDANDO_VAULT_ORGANIZADO
2. VERIFICAR MC-MAPA-Documentacao-vX_Y.md existe no Vault → SE NÃO: ABORTAR + sinalizar AGUARDANDO_DOCUMENTACAO_ESTRUTURADA
3. VERIFICAR MC-DOCUMENTACAO-INSTRUCAO-MARCA-vX_Y.md existe no Vault → SE NÃO: ABORTAR + sinalizar AGUARDANDO_DOC_INSTRUCAO_MARCA_C1
4. VERIFICAR pasta Identidade Visual\01-Brand-Foundation-v1_0\ existe com paleta + tipografia + logos → SE NÃO: ABORTAR + sinalizar AGUARDANDO_BRAND_FOUNDATION_C5_FOUNDER
5. SE 4 verificações OK: prosseguir com produção visual em massa conforme MB recebido
```
