# MC-PROMPT-Aplicacoes-GPT v1.0

**Biblioteca canônica de prompts derivados por aplicação específica.**

Cada prompt abaixo é uma adaptação contextual do `MC-PROMPT-MetaCanonico-Renderizacao-v1_0-2026-0513.md`. **O Seed MC, a paleta cósmica + ouro, a atmosfera, a tipografia e o vocabulário canônico são preservados integralmente em todas as aplicações.** Apenas o formato, microtexto contextual e densidade da composição se adaptam.

**Princípio inviolável:** se ao adaptar uma aplicação tu sente vontade de "modernizar" o Seed, "deixar mais leve", "simplificar para mobile", "adaptar para o público X" — **pare**. A adaptação é apenas contextual (formato/microtexto/densidade). Símbolo, paleta, atmosfera e tipografia são imutáveis.
