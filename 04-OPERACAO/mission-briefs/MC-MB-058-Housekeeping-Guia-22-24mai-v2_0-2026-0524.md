### Fase 4 · Hash-verify pós-Copy-Item (#36)

```powershell
$baseline = Get-Content "$transit\baseline-mb058-v2_0.json" | ConvertFrom-Json
$violacoes = 0

foreach ($arq in $mapeamento.Keys) {
    $destino = "$($mapeamento[$arq])$arq"
    $hash_destino = (Get-FileHash $destino -Algorithm SHA256).Hash

        Write-Host "❌ Hash violado: $arq" -ForegroundColor Red
```
